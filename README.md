# Powerbi-superstore-sales-analysis-dashboard
# Overview :- This project  analyzes Sales performance across regions, products, and time to help management make data driven decisions.
# Tools Used:-
  - Power Bi
  - Excel
  - DAX
  - Power query
# Dataset:-
  - Order ID,	Order Date,	Ship Date, Ship Mode, Customer Name, Segment,	Country,	City,	State,	Region,	Product ID,	Category,	Sub-Category,	Product Name,	Sales,	Quantity,	Profit,	Returns,	Payment Mode.
